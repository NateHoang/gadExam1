Chromosomal / Microdeletion
| Syndrome | Category | Key features |
|---|---:|---|
| **Down syndrome (Trisomy 21)** [page 51] | Chromosomal (Trisomy 21) | Intellectual disability, hypotonia, characteristic facies, cardiac defects; oral findings: Class III malocclusion, narrow palate, hypodontia. |
| **Patau syndrome (Trisomy 13)** [page 9] | Chromosomal (Trisomy 13) | Severe CNS defects (holoprosencephaly), polydactyly, microcephaly; often lethal. |
| **Cri‑du‑chat (5p deletion)** [page 10] | Chromosomal deletion | High‑pitched cry, failure to thrive, hypotonia, intellectual delay. |
| **Turner syndrome** [page 11] | Sex chromosome (45,X or mosaic) | Short stature, streak ovaries, lymphedema, neck webbing, congenital cardiac/renal defects. |
| **Klinefelter syndrome (XXY)** [page 12] | Sex chromosome (extra X) | Hypogonadism, infertility, tall stature, feminized features. |
| **DiGeorge syndrome (22q11.2 deletion)** [page 15] | Microdeletion (22q11.2) | Cleft palate, congenital heart disease, thymic hypoplasia → T‑cell dysfunction, hypocalcemia (parathyroid). |
| **Prader‑Willi syndrome** [page 13] | Imprinting disorder (paternal loss) | Infant hypotonia/poor feeding; childhood hyperphagia, obesity, hypogonadism, intellectual disability. |
| **Angelman syndrome** [page 14] | Imprinting disorder (maternal loss) | Severe intellectual disability, seizures, ataxia, inappropriate laughter. |

Mendelian (single‑gene) — Autosomal dominant / recessive / X‑linked
| Syndrome | Category | Key features |
|---|---:|---|
| **Apert syndrome** [page 53] | Mendelian — Autosomal dominant (craniosynostosis) | Premature suture fusion, syndactyly, midface hypoplasia, dental crowding, possible cleft palate. |
| **Crouzon syndrome** [page 57] | Mendelian — Autosomal dominant (craniosynostosis) | Midface hypoplasia, ocular proptosis, hypertelorism, variable skull shapes. |
| **Achondroplasia** [page 18] | Mendelian — Autosomal dominant | FGFR3 mutation → disproportionate short stature (dwarfism). |
| **Cleidocranial dysplasia** [page 19] | Mendelian — Autosomal dominant | Clavicle hypoplasia/aplasia, open sutures, supernumerary teeth, delayed eruption. |
| **Cherubism** [page 21] | Mendelian — Autosomal dominant | Bilateral jaw swelling, multilocular radiolucencies, dental displacement. |
| **EEC syndrome (Ectrodactyly‑Ectodermal Dysplasia‑Clefting)** [page 22] | Mendelian — variable inheritance | Lobster‑claw deformities (ectrodactyly), ectodermal defects (hair/nails/teeth), cleft lip/palate. |
| **Ataxia‑Telangiectasia** [page 24] | Mendelian — Autosomal recessive | ATM mutation → cerebellar ataxia, telangiectasias, immunodeficiency, cancer risk. |
| **Fragile X syndrome** [page 25] | X‑linked (CGG repeat expansion) | Intellectual disability (males), long narrow face, large ears, macroorchidism. |
| **Hunter syndrome (MPS II)** [page 28] | X‑linked recessive (lysosomal storage) | Iduronate sulfatase deficiency → GAG buildup, coarsened facies, hepatosplenomegaly, airway/cardio issues. |
| **Hurler syndrome (MPS I)** [page 26] | Autosomal recessive (lysosomal storage) | IDUA mutation → severe MPS: coarse facies, corneal clouding, organomegaly, short life expectancy. |
| **Papillon‑Lefèvre syndrome (PLS)** [page 75] | Mendelian — Autosomal recessive | CTSC mutation → palmoplantar hyperkeratosis, early aggressive periodontitis, tooth loss. |
| **Gardner syndrome** [page 77] | Mendelian — Autosomal dominant (APC gene) | Hundreds of colon polyps (cancer risk), extra teeth, osteomas/desmoid tumors. |

Craniofacial / Neural crest (neurocristopathies) & Sequence
| Syndrome / Sequence | Category | Key features |
|---|---:|---|
| **Pierre Robin sequence** [page 47] | Sequence (mandibular micrognathia → glossoptosis) | Micrognathia → downward tongue → cleft palate, airway/feeding issues. |
| **Van der Woude syndrome** [page 49] | Craniofacial (autosomal dominant) | Cleft lip/palate with paramedian lower lip pits. |
| **Treacher‑Collins syndrome** [page 62] | Neurocristopathy — Autosomal dominant | Hypoplastic zygomas, mandibular hypoplasia, downward slanting palpebral fissures, hearing loss (conductive). |
| **Holoprosencephaly (HPE)** [page 31] | Developmental malformation / syndromic cause | Forebrain midline failure → facial dysmorphism (cyclopia to mild forms), severe neuro impairment. |
| **DiGeorge** (also neurocristopathy) [page 15] | (see above) | See DiGeorge entry. |

Connective tissue / skeletal disorders
| Syndrome | Category | Key features |
|---|---:|---|
| **Ehlers‑Danlos syndrome** [page 65] | Connective tissue disorder (various inheritance) | Hyperextensible skin, hypermobile joints, easy bruising; types IV & VIII → severe periodontal disease, aortic risk. |
| **Marfan syndrome** [page 69] | Connective tissue — Autosomal dominant | Tall, arachnodactyly, aortic root dilation/aneurysm risk, high‑arched palate, TMJ issues. |
| **Osteogenesis imperfecta** [page 71] | Collagen defect (various inheritance) | Fragile bones, blue sclerae, dentinogenesis imperfecta, hearing loss. |

Other / Vascular / Inflammatory / Non‑genetic
| Syndrome | Category | Key features |
|---|---:|---|
| **Progressive hemifacial hypoplasia (Parry‑Romberg syndrome)** [page 73] | Non‑genetic, sporadic | Unilateral facial atrophy, coup de sabre scleroderma, alopecia, delayed dental development. |
| **Kawasaki disease** [page 79] | Pediatric vasculitis (likely infectious trigger) | Fever, rash, mucocutaneous changes, coronary artery inflammation → aneurysm risk. |
